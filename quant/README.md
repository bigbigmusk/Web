# 📈 量化交易工具（小白友好版）

> 一套用来**分析股票和加密货币**、**回测交易策略**、**生成实时买卖信号**的工具。
> 纯 Python 标准库写成，**零依赖、装好 Python 就能跑**。专为零基础新手设计。

---

## ⚠️ 先读这一段（非常重要）

1. **这是分析工具，不是印钞机。** 它帮你用数据做决策，不能保证赚钱。
2. **市场有风险，可能亏光本金。** 尤其是币圈，波动极大，杠杆合约能让你几小时归零。
3. **本工具不构成任何投资建议。** 所有买卖决定和后果由你自己承担。
4. **先学会，再小钱实盘，永远不要 All in。** 单笔亏损控制在本金的 1~2% 以内。
5. **本工具默认只做“分析和回测”，不会自动帮你下单交易**（自动下单需要你自己接交易所 API，风险自负）。

---

## 🧠 三个核心概念（30 秒看懂量化）

| 概念 | 大白话解释 |
|------|-----------|
| **技术指标** | 把价格做数学加工，帮你“看图说话”，比如均线、RSI、MACD。 |
| **交易策略** | 一套固定的买卖规则，比如“短期均线上穿长期均线就买”。 |
| **回测** | 拿历史数据假装“当年照这套规则做”，看最后是赚是亏。**这是检验策略的第一关。** |

> 一句话：**先想规则 → 用历史数据回测 → 跑赢基准且回撤可控 → 再用小钱实盘验证。**

## 🚀 三步上手

### 第 1 步：装 Python（只需一次）
- 去 [python.org/downloads](https://www.python.org/downloads/) 下载安装 Python 3.9+
- Windows 安装时**务必勾选** “Add Python to PATH”
- 装好后打开“终端”（Mac）或“命令提示符 / PowerShell”（Windows），输入 `python3 --version` 能看到版本号就成功了

### 第 2 步：进入本文件夹
在终端里 `cd` 到这个 `quant` 文件夹。例如：
```bash
cd 路径/到/quant
```

### 第 3 步：运行
```bash
python3 run.py
```
然后照着菜单输入数字就行。**第一次建议先选 `4`（离线演示）**，不用联网就能把整个流程跑通、看明白。

## 📋 菜单功能说明

```
1. 回测单个策略      → 选市场、品种、策略，看它历史上赚不赚、回撤多大
2. 策略横向对比      → 同一个品种，把所有策略跑一遍排排坐，一眼看出谁强
3. 实时信号扫描      → 现在这个币/股，各策略给的是“买入/观望/卖出”
4. 离线演示          → 不用联网，用模拟数据先体验全流程
0. 退出
```

回测和对比结束后，会在 `quant/reports/` 里生成一个 **HTML 可视化报告**，
用浏览器打开就能看到：价格走势 + 买卖点、资金曲线对比、各项绩效指标。

## 💻 命令行用法（进阶，可跳过）

```bash
python3 run.py demo                             # 离线演示
python3 run.py backtest crypto BTCUSDT ma_cross # 回测比特币的双均线策略
python3 run.py backtest us AAPL macd            # 回测苹果的 MACD 策略
python3 run.py backtest cn sh600519 boll        # 回测贵州茅台的布林带策略
python3 run.py compare crypto ETHUSDT           # 以太坊：所有策略横向对比
python3 run.py scan crypto                      # 扫描比特币/以太坊/SOL 当前信号
```

**市场代码**：`crypto`=币圈，`us`=美股，`cn`=A股，`csv`=本地文件
**品种代码示例**：
- 币圈：`BTCUSDT`、`ETHUSDT`、`SOLUSDT`（币安交易对格式）
- 美股：`AAPL`、`NVDA`、`TSLA`、`SPY`、`QQQ`
- A股：`sh600519`（茅台）、`sz000001`（平安银行）、`sh000001`（上证指数）

**策略代码**：`ma_cross`（双均线）、`rsi`、`macd`、`boll`（布林带）

## 📊 怎么看回测报告（看懂这几个就够了）

| 指标 | 含义 | 怎么算好 |
|------|------|---------|
| **总收益率** | 这段时间一共赚了百分之几 | 越高越好，但要看回撤 |
| **买入持有基准** | 啥都不做、一直拿着的收益 | 策略**必须跑赢它**才有意义 |
| **最大回撤** | 从最高点最多跌了多少 | 越小越好，>30% 普通人扛不住 |
| **夏普比率** | 每承担 1 份风险换来多少收益 | >1 不错，>2 优秀 |
| **胜率** | 赚钱的交易占比 | 高不一定好，要配合盈亏比看 |
| **盈亏比(PF)** | 总盈利 ÷ 总亏损 | >1.5 较好 |

> 划重点：**收益高但回撤巨大 = 危险**。稳定地小赚、回撤可控，远胜大起大落。

## 🗂️ 用自己的数据（CSV）

任何行情软件（同花顺、TradingView、交易所）导出的 CSV 都能用，只要有这几列表头：
`date, open, high, low, close, volume`（大小写无所谓，中文“日期/开盘/收盘…”也认）。

```bash
python3 run.py
# 选 1 → 市场选 4（CSV）→ 输入文件路径，如 examples/sample.csv
```

`examples/sample.csv` 是自带的示例数据，离线时也能用它练手。

## ❓ 常见问题

**Q：运行时提示“数据获取失败”？**
A：可能是网络问题或该数据源在你所在地区受限。工具会自动切换备用源；如果都失败，会用离线模拟数据让你先跑通流程。也可以用 CSV 导入自己的数据。

**Q：A股 / 港股能做空吗？**
A：A股普通账户不能裸做空，所以工具对 A股默认只做多。币圈和美股可以做空，但新手强烈建议先只做多。

**Q：回测赚翻了，实盘能照搬吗？**
A：**千万别。** 回测最容易“过拟合”（专门挑出历史上最好看的参数）。务必先用很小的钱实盘验证至少几周。

**Q：怎么加我自己的策略？**
A：打开 `lib/strategy.py`，复制一个现有策略函数改规则，再在底部 `REGISTRY` 里登记一行即可。

1. 先用**离线演示**和**示例 CSV**把流程玩熟
2. 换不同品种、不同策略做**横向对比**，建立盘感
3. 学习**仓位管理**和**止损**（比选策略更重要）
4. 想真正实盘，再研究交易所 API（币圈推荐 `ccxt` 库）——但请先确保你已经能稳定盈利

> 记住：**活下来比赚得快重要一百倍。** 控制风险，慢慢变富。
